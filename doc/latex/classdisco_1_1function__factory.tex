\doxysubsection{Detailed Description}
Utility class, used to create some sorts of functional object wrappers. 

Allowed function types are\+:
\begin{DoxyItemize}
\item pointer to member function, the object on which the method will be called is also required
\item pointer to function or static method
\item raw lambda function or functional object (operator() has to be declared)
\item std\+::function$<$\+TSignature$>$ objects 
\end{DoxyItemize}
